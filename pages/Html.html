Common Elements

HTML elements are classified to void element and normal element.

Heading elements from h1 to h6 used to signify the importance of content below them.

p element is used to create the a paragraph of text on websites

Comment is allowed by the html by use <!–, -->

main element is used to represent the main content of the body of an HTML document

img element is used to linked with a photo.

a element is used to be linked with another link. Other types of content can also be turned into a link by wrapping it in anchor tags.

section element is used to define sections in a document

ul element can be used to create an unordered list of items

li element is used to create a list item in an ordered or unordered list. Almost nested by ul

figure element represents self-contained content and will allow you to associate an image with a caption.

ol element is similar to an unordered list, but list items in an ordered list are numbered when displayed.

figcaption is used to add a caption to describe the image contained within the figure element.

em element alows us to place emphasis on a specific word or phrase

strong element is used to indicate that some text is of strong importance or urgent.

form element is used to get information from a user like their name, email, and other details.

Common Attributes

HTML attributes are special words used inside the opening tag of an element to control the element’s behavior.

The src attribute in an img element specifies the image’s URL (where the image is located).

All img elements should have an alt attribute. The alt attribute’s text is used for screen readers to improve accessibility and is displayed if the image fails to load.

href attribute is linked with a URL.

Target attribute specifies where to open the linked document.

action attribute indicates where form data should be sent (to form).
